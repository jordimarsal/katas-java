Endpoints

Location of the endpoints in this application.

  • /users/
  • /katas
  • API HTML ui: /swagger-ui.html
  • API Definition: /v3/api-docs/
  • H2 Database: /h2-console/

CURL Operations:

> curl -X GET http://localhost:9090/users/users
> curl -X GET http://localhost:9090/users/jordi
> curl -X PUT -H "Content-Type: application/json" \
  -H "Key: Magic_Key" \
  -d '{"firstName":"Jordi","lastName":"M"}' \
  http://localhost:9090/users/users/jordi

> curl -X GET http://localhost:9090/katas/all
> curl -X GET http://localhost:9090/katas/Kata 1

> curl -X GET http://localhost:9090/users/jordi

> curl -X GET http://localhost:9090/v3/api-docs/
